Introduction

"Hello, World" was first seen in the book: A Tutorial Introduction to the Programming Language B. The book was written by Brain Kernighan and published in 1978. The author has stated he can't remember when or why he choose those the saying "Hello, World". The "Hello, World" program is usually the first program written when learning a programming language. The program is simple and is written so the text "Hello, World" is displayed somewhere on the screen for the user to see.

Top 10 languages on Github in 2018

According to Octoverse, the annual survey that GitHub conducts, the top ten languages on GitHub in 2018 were:

  1. Javascript
  2. Java
  3. Python
  4. PHP
  5. C++
  6. C#

  8. Shell
  9. C
  10. Ruby
Javascript

JavaScript (which has no actual relation to Java), is a scripting language developed originally for use in web browsers. Because of that, it has a built-in facility for dealing with the Document Object Model, the in-memory representation of the content of a webpage. It is the primary programming language for front-end web development.

// Hello world in JavaScript

console.log("Hello World");
Java

Java is a high-level language designed to be used by the Java Virtual Machine. It has very few external dependencies, and was intended to be able to run on any physical machine. It is used a lot in network architecture and also in embedded devices, kiosks, and other in situ computing applications.

// Hello World in Java

class HelloWorld {
 static public void main( String args[] ) {
  System.out.println( "Hello World!" );
 }
}
Python

Python is a high-level programming language. It is an interpreted (not compiled) language, also known as a “scripting language.” It is mostly used as a tool for performing ad hoc programming tasks such as task automation and data analysis. Python 3 is the latest iteration of the Python language.

# Hello world in Python 3 (aka Python 3000)

print("Hello World")
PHP

PHP is the most popular server-side language on the internet today. It is incredibly powerful yet easy to learn.

<?php
 // Hello world in PHP
 echo 'Hello World!';
?>
C++

Originally “C with Classes,” C++ is, in many ways, simply a more advanced successor to C (though the situation is a lot more complicated than that). C++ was developed to add high-level programming paradigms to C, while retaining the low-level hardware-manipulation capabilities.

// Hello World in C++

#include <iostream>
using namespace std;
int main()
{
 cout << "Hello World!" << endl;
}
C#

Used as the primary language for .NET programming, and much like C++, it is an extension of the C programming language with the major addition being object-oriented capabilities.

//Hello World in C#
class HelloWorld
{
 static void Main()
 {
  System.Console.WriteLine("Hello, World!");
 }
}
TypeScript

Microsoft TypeScript is an open-source programming language developed and maintained by Microsoft. It is a strict syntactical superset of JavaScript, and adds optional static typing to the language. TypeScript is designed for development of large applications and transcompiles to JavaScript.

// Hello World in TypeScript

alert('Hello world!');
Shell

A shell program (sometimes called a shell script) is a text file that contains standard UNIX and shell commands.

#Hello World in Shell
#!/bin/sh
echo Hello world!
C

If you include its two derivatives, no language has had more use and more impact than C. It is especially important for the development of Operating Systems and other foundational software. Many compilers and interpreters for other languages are written in C.

//Hello World in C
#include <stdio.h>
int main()
{
 printf("Hello, World!");
 return 0;
}
Ruby

Ruby is a popular programming language. It is particularly associated with Rails, a web development framework for the Ruby programming language. It provides an MVC (Model View Controller) architecture, a database abstraction layer, and a lot of tools to speed the process of web application programming.

# Hello World in Ruby
puts "Hello World!"
